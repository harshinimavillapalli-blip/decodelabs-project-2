This Python program is a simple Expense Tracker that allows a user to enter multiple expense amounts and calculates the total money spent. It demonstrates important programming concepts such as variables, loops, user input, conditional statements, exception handling, and accumulators.

# Expense Tracker Application

A simple command-line based Expense Tracker built using Python. This project allows users to enter multiple expense amounts, calculates the total spending, and displays a summary at the end.

# Features

- Add multiple expense entries
- Automatically calculates total spending
- Simple and beginner-friendly interface
- Runs continuously until the user chooses to finish
- Demonstrates the accumulator concept in Python

# Technologies Used

- Python

# How It Works

The program starts with a total expense value of 0.

Users can:

- Enter expense amounts one by one
- Type `done` when finished entering expenses

The program adds each expense to the running total and displays the final amount spent.

# How to Run

1. Install Python on your system.
2. Save the code in a file named `expense_tracker.py`
3. Open Terminal or Command Prompt.
4. Run the program using:

```bash
python expense_tracker.py
```
